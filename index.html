🔍 วิธีแก้ไขหากไม่สามารถเปิดกล้องได้:

  1. อนุญาตการเข้าถึง: คลิก "Allow" หรือ "อนุญาต" เมื่อเบราว์เซอร์ถามสิทธิ์
  2. ตรวจสอบ URL: URL ต้องขึ้นต้นด้วย https:// (มีตัว s)
  3. ปิดโปรแกรมอื่น: ปิด Zoom, Teams, Skype, หรือโปรแกรมที่ใช้กล้อง
  4. ตรวจสอบ Settings:
    • Chrome: Settings → Privacy and security → Site Settings → Camera
    • Firefox: Preferences → Privacy & Security → Permissions → Camera
    • Safari: Safari → Preferences → Websites → Camera
  5. รีสตาร์ทเบราว์เซอร์: ปิดเบราว์เซอร์ทั้งหมดแล้วเปิดใหม่
  6. เปลี่ยนเบราว์เซอร์: ลองใช้ Google Chrome หรือ Microsoft Edge
🔒 ความเป็นส่วนตัว: ข้อมูลวิดีโอจะถูกประมวลผลในเครื่องของคุณเท่านั้น
